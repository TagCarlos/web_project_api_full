# Tripleten web_project_api_full

Descipción: pagina web que simula una red social donde puedes interactuar, registrandose, iniciar sesion, compartir imagenes darles like, colocar un avatar y una breve descripción del usuario.

tecnologias utilizadas: para el backend celebrate, cors, express, express-winston, moongose, validator, winston

para el frontend react, eslint, vite
